| resources | 文件夹 | 存放资源文件 |
| .gitignore | 文件 | git忽略文件 |
| LICENSE | 文件 | 许可证 |
| README.md | 文件 | 项目说明文档 |
| env.py | 文件 | 游戏环境文件 |
| dqn.py | 文件 | DQN算法文件 |
| minst_data.py | 文件 | 数据集文件 |
| traint.py | 文件 | 训练文件 |
| requrements.txt | 文件 | 环境依赖文件 |
| play.py | 文件 | 运行文件 |
| hwnd.py | 文件 | 窗口句柄 |
